PXCheckoutPreference

@objcMembers
open class PXCheckoutPreference : NSObject, Codable

Model that represents curl -X OPTIONS (https://api.mercadopago.com/checkout/preferences) | json_pp It can be not exactly the same because exists custom configurations for open Preference. Some values like: binary mode are not present on API call.
